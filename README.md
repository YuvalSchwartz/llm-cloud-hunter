## Running instructrions

To run LLM-Cloud-Hunter, use the main.py file.
Add url's of new OSCTI to the urls list

### Example:
```python
if __name__ == '__main__':
    urls = []
    urls.append('https://sysdig.com/blog/cloud-breach-terraform-data-theft/')
```
